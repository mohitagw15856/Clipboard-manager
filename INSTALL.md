# Quick Installation Guide

## Step-by-Step Installation

### 1. Download the Extension
Download all files in the `clipboard-manager` folder to your computer.

### 2. Open Chrome Extensions Page
- Open Google Chrome
- Type `chrome://extensions/` in the address bar
- Press Enter

### 3. Enable Developer Mode
- Look for the "Developer mode" toggle in the top-right corner
- Click it to turn it ON

### 4. Load the Extension
- Click the "Load unpacked" button (appears after enabling Developer mode)
- Navigate to and select the `clipboard-manager` folder
- Click "Select Folder" or "Open"

### 5. Verify Installation
- You should see "Multi-Clipboard Manager" in your extensions list
- The extension icon should appear in your Chrome toolbar (puzzle piece icon)
- If you don't see it in the toolbar, click the puzzle piece and pin it

### 6. Test It Out
1. Click the extension icon
2. Type some text in Slot 1
3. Click "Save"
4. Go to any text field on any website
5. Press `Ctrl+Shift+1` (or `Cmd+Shift+1` on Mac)
6. Your text should be pasted!

## Quick Start Usage

1. **Save Text**: Click extension icon → Type text → Click "Save"
2. **Paste Text**: Press `Ctrl+Shift+[1-5]` while in any text field
3. **Customize Shortcuts**: Click "Customize Shortcuts" in the popup

## Default Shortcuts

- `Ctrl+Shift+1` - Paste Slot 1
- `Ctrl+Shift+2` - Paste Slot 2
- `Ctrl+Shift+3` - Paste Slot 3
- `Ctrl+Shift+4` - Paste Slot 4
- `Ctrl+Shift+5` - Paste Slot 5

(Use `Cmd` instead of `Ctrl` on Mac)
